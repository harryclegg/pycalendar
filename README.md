# pycalendar
Generate a PDF of a monthly calendar for printing.

## Usage

Install dependencies:

```
numpy
Pillow
reportlab
frontend
pymupdf
```

Then run the script using python 3.8.

It will generate one PDF per month for the current (or upcoming, if close to New Year) year.
